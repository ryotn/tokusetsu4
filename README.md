# For developers

## Set up

```
$ git clone git@github.com:sanographix/tokusetsu4.git tokusetsu4
$ cd tokusetsu4
$ npm install
```

## Start local server

```
$ npm start
```

Default URL is <http://localhost:3000/>.

## Build

```
$ npm run build
```

Assets will output to `build/`.
